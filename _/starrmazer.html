Starr Mazer

A retro-sexy point-and-click adventure SHMUP set in space.

PROJECT INFO

ROLES Lead Programmer, Lead Designer
PLATFORMS PC
RELEASE DATE TBA
START DATE January 2015
TEAM SIZE 7-10

KEY CONTRIBUTIONS

BUILT ​MAZER MAKER
Q visual editor that empowers non technical team members to implement, iterate, and test their content, be it writing, animation, audio, cutscenes, action gameplay, items, puzzles, etc.

DEVELOPED UNITY TO STEAM BUILD TOOL
to push game builds to steam with one button or automated through the command line

IMPLEMENTED DYNAMIC EVENT SYSTEM
based on a Valve GDC talk that allows responsive and dynamic game content that triggers functionality across the codebase without tight coupling

REDESIGNED AND BLENDED TWO DISPARATE GENRES
Modern design is brought to both PnC and SHMUP genres to help combine the two into a cohesive experience. Slow paced pixel hunting is removed with object outlining. Lock-and-key puzzles are eased with multi-use items and multiple solutions. SHMUP gameplay is slowed down with character dialogue and PnC interactions. SHMUP difficulty is toned down with regenerative health and narrative consequences replacing game overs.

ASSISTED IN A SUCCESSFUL ​KICKSTARTER CAMPAIGN
Ended with 5,055 backers contributing a total of $193,566
