CatDate

CatDate is a game where you play as a cat and go on dates with other cats. It's a project which focuses on communicating skills in empathy through meaningful and hilarious writing with a tight game loop.

PROJECT INFO

ROLES Designer, Programmer, Artist
PLATFORMS iOS, Android
RELEASE DATE TBA
START DATE January 2016
TEAM SIZE 2

KEY CONTRIBUTIONS

ACCEPTED AT STUGAN
Helped with the pitch that got the game accepted as one of Stugan's pick for a 7 week game development accelerator in Sweden.

MERGED MOBILE AND VISUAL NOVEL DESIGN
Contributed to the design of a tight system which encourages play beyond typical visual novels to keep players engaged beyond the story.

BUILT CUSTOM DIALOGUE TOOL
Developed the Cat Chat Creator, a narrative tool built in Unity in order for the writer to create branching and responsive dialogue quickly within the game engine.

BUILT DIALOGUE MARKUP LANGUAGE
Developed a markup language for our dialogue to allow the writer to trigger game code at will inline with the dialogue.

ART ASSETS AND DIRECTION
Cooperated in the creation of a simple but cute and approachable art style.

BUILT SIMPLE ART PIPELINE
Created a pipeline for implementing assets into our game which prepares them for animation and avatar customization.
